Reiniciar Sistema HR Organic

Esta herramienta te permite reinicializar todos los datos del sistema, incluyendo el nuevo sistema de usuarios.

ADVERTENCIA: Esta acción eliminará TODOS los datos almacenados localmente, incluyendo empleados, nóminas, recolecciones, etc. No se puede deshacer.

Después de reiniciar se creará:

  • Usuario admin: admin / 1234
  • Rol: Administrador (acceso completo)
  • Datos de ejemplo: Empleados, departamentos, cargos, etc.
  • Módulo de usuarios: Gestión de Usuarios activo
